# 00-core – Zentrale Basis des Project Manu Core

Der Ordner `00-core` ist das Fundament des gesamten Repos.  
Hier liegt alles, was **für alle anderen Ordner gilt**:

- Grundprinzipien
- Konventionen
- Regeln für KI-Nutzung
- Struktur- und Benennungslogik
- Vorlagen für neue Inhalte
- Strategien für Kontext- und Dateiauswahl

Wenn eine KI oder ein Mensch verstehen will, **wie dieses Repo gedacht ist**,  
beginnt sie hier.

**Wichtig:**  
Alle anderen Ordner (01–04) bauen auf den Regeln und Rahmenbedingungen aus `00-core` auf.
